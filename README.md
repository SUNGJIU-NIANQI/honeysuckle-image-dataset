# 金银花与山银花图像数据集

基于 CNN 卷积神经网络的金银花与山银花鉴别模型训练用图像数据集。

## 数据来源

所有图片均在实验室环境下拍摄获取，为未经过预处理的原始图像。

## 目录结构

```
├── train/              # 训练集
│   ├── jinyinhua/      #   金银花
│   └── shanyinhua/     #   山银花
├── test/               # 测试集
│   ├── jinyinhua/      #   金银花
│   └── shanyinhua/     #   山银花
└── remix/              # 混合数据集
```

## 数据概况

| 类别 | 说明 |
|------|------|
| 图片数量 | 1348 张 |
| 图片格式 | JPG |
| 拍摄环境 | 实验室实拍 |
| 预处理状态 | 原始图像，未做预处理 |

## 用途

用于训练和评估基于卷积神经网络（CNN）的金银花与山银花图像分类鉴别模型。
